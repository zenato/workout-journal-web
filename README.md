# Workout journal

React based workout journal web client toy project.

## Getting Started

### Install dependency modules
`yarn`

### Start app
`yarn run start`

### Test app
`yarn run test`

### Production build
`yarn run build`

You may need to modify `packages.json`. (homepage, proxy settings..)
